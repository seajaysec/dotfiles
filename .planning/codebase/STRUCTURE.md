## Key Files

### Shell Configuration

| File | Purpose |
|---|---|
| `.zshrc` | Primary shell config. Sets environment variables, PATH, history, loads Zap plugin manager, sources aliases/functions/secrets, configures Starship prompt, sets up vi mode. Deployed to `~/.zshrc` via copy. |
| `.zsh.aliases` | All shell aliases organized by category. Sourced from `~/dotfiles/.zsh.aliases` at shell startup. Covers system commands, navigation, networking, git, docker, python, and text processing. |
| `.zsh.functions` | All custom shell functions. Sourced from `~/dotfiles/.zsh.functions`. Contains networking utilities (IP extraction, CIDR expansion, dual-ping), security tools (cert inspection, CVE scanning), file operations, git helpers, and media conversion. |
| `completions.zsh` | Configures zsh completion system: cache, grouping, formatting, menu selection. Sourced before `compinit` in `.zshrc`. |
| `.fzf.zsh` | Legacy fzf setup for `/usr/local/opt/fzf`. Currently superseded by `source <(fzf --zsh)` in `.zshrc`. |
| `.p10k.zsh` | Powerlevel10k prompt configuration generated 2021-11-29. Legacy — no longer sourced since Starship replaced it. Kept for reference. |

### Tmux Configuration

| File | Purpose |
|---|---|
| `.tmux.conf` | gpakosz/.tmux base framework (1484 lines). Provides general settings, navigation keybindings, copy-mode setup, and an embedded shell script for dynamic theming. The comment block after `# EOF` contains the actual theme engine. **Do not edit this file directly.** |
| `.tmux.conf.local` | User customizations sourced by `.tmux.conf`. Contains: Dracula-inspired color theme, window/pane behavior flags, status bar layout, clipboard integration, mouse mode, and TPM plugin declarations (12 plugins including resurrect, continuum, logging, yank, extrakto). |

### Installation & Maintenance

| File | Purpose |
|---|---|
| `install.sh` | One-shot bootstrap for a fresh macOS machine. Installs Xcode CLI tools, Homebrew, 22 brew packages, NVM, Bun, Rust, copies config files, sets zsh as default shell. |
| `brewup.sh` | Recurring maintenance script. Updates brew packages, dumps installed packages to a hostname-tagged Brewfile, commits and pushes to the repo. Handles Apple Silicon PATH differences. |
| `Brewfile.C02FR3U9MD6T` | Auto-generated Homebrew bundle dump for a specific machine. Contains 465 lines of taps, brews, casks, and mas entries. Filename includes the machine's hostname. |

### Prompt Configuration

| File | Purpose |
|---|---|
| `config/starship/starship.toml` | Starship cross-shell prompt config (252 lines). Uses gruvbox-dark palette with powerline-style segments. Shows: sudo indicator, username, directory, git branch/status, Python virtualenv, package version, command duration, and exit status with table-flip emoji (`\(╯°□°）╯︵ ┻━┻`). |

### SwiftBar / xbar Menu Bar Scripts

| File | Purpose |
|---|---|
| `swiftbars/mac-mux.sh` | Lists active tmux sessions in the macOS menu bar. Click to attach to a session or create a new one. |
| `swiftbars/slack-status.sh` | Set Slack status from the menu bar via Autocode API. Presets for focus, meeting, break, lunch, done, vacation. |
| `swiftbars/tail.5s.sh` | Tails a configured log file in the menu bar, refreshing every 5 seconds. |
| `swiftbars/zoom.1s.scpt` | AppleScript that checks Zoom meeting status, updating every 1 second. |

### Git Configuration

| File | Purpose |
|---|---|
| `.gitignore` | Repo-level: ignores `.DS_Store` only. |
| `.gitignore_global` | Global gitignore template covering macOS system files, Python artifacts, virtualenvs, Vim swap files, VSCode workspace files. Intended for `git config --global core.excludesfile`. |

### Terminal Exports

| File | Purpose |
|---|---|
| `iTermProfiles.json` | iTerm2 profiles export (JSON). |

## Naming Conventions

### Dot-Prefix Pattern

Shell configuration files use a consistent dot-prefix naming scheme:

- **`.zshrc`** — main zsh runtime config
- **`.zsh.aliases`** — zsh aliases (dot-separated namespace: `.zsh.{concern}`)
- **`.zsh.functions`** — zsh functions (same pattern)
- **`.tmux.conf`** — tmux base config
- **`.tmux.conf.local`** — tmux overrides (`.{tool}.conf.local` for override layer)
- **`.fzf.zsh`** — fzf-specific zsh integration
- **`.p10k.zsh`** — powerlevel10k config (legacy)
- **`.gitignore`** / **`.gitignore_global`** — git ignores (repo vs global scope)

### Config Subdirectory Pattern

Non-shell application configs go under `config/{appname}/`:
- `config/starship/starship.toml`

### Script Naming

- **SwiftBar/xbar scripts** follow the xbar naming convention: `{name}.{interval}.{ext}` (e.g., `tail.5s.sh` runs every 5 seconds, `zoom.1s.scpt` runs every 1 second)
- **Maintenance scripts** use descriptive names: `install.sh`, `brewup.sh`

### Brewfile Naming

Machine-specific Brewfiles use the pattern `Brewfile.{HOSTNAME}` (e.g., `Brewfile.C02FR3U9MD6T`), auto-generated by `brewup.sh`.

## Platform-Specific Organization

### macOS Focus

This dotfiles repo is macOS-specific. Key platform assumptions:

- **Homebrew** at `/opt/homebrew` (Apple Silicon) with fallback to `/usr/local` (Intel)
- **pbcopy/pbpaste** for clipboard operations throughout aliases and functions
- **macOS-specific commands**: `diskutil`, `pmset`, `osascript`, `dscacheutil`, `blueutil`
- **iTerm2** integration (shell integration script, profile exports)
- **SwiftBar/xbar** menu bar plugins
- **Finder integration**: `f` alias jumps to current Finder window, `remove_dups` navigates via Finder
- **`ARCHFLAGS="-arch x86_64"`** default with `ssdeep_env()` function for ARM64 builds

### Kali Linux (Removed)

The `Kali/` directory previously contained Linux-specific variants of the dotfiles (`.tmux.conf.local`, `.zsh.aliases`, `.zsh.functions`, `.zshrc`, plus a `README.md` and `install.sh`). Per the current git status, all Kali files have been **deleted** — the repo is now macOS-only.

### Apple Silicon Awareness

`brewup.sh` detects architecture via `$(arch)` and adjusts the Homebrew path:
- ARM64 (Apple Silicon): `/opt/homebrew/bin:/opt/homebrew/sbin`
- x86_64 (Intel): `/usr/local/bin` (default)

The `.zshrc` PATH configuration uses `/opt/homebrew/bin` as the primary path, indicating the primary machine is Apple Silicon.
